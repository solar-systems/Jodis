# Jodis(Java Object Dictionary Server)
Jodis is keyvalue db, the goal is to achieve most standalone feature for [Redis](https://github.com/redis-io/redis).

## Feature Support
### in-memory key-value database
1. JodisString
      
2. JodisList

3. JodisMap

4. JodisSet

5. JodisZSet
